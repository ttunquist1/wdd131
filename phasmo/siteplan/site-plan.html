Overview

Purpose

Phasmophobia is a fun paranormal investigation game, where you and a group of three friends are able to use your investigation abilites to figure out which of 24 unique ghost are hunting the house. This site is meant to be a companion to the game, where you will be able to track the evidences and and see the unique behaviors of each ghost type.

Audience

This site is intended for beginners to the game and for the experienced player as well. This site will help all to be able to better track the ghost and learn how to differentiate the ghosts themselves.

Dynamic elements

I will have each ghost loaded in a json style, and dynamically change which ones appear on the screen depending on which evidences or behaviors have been selected. The evidences will act as a filter to decied which ghosts are displayed on screen.
